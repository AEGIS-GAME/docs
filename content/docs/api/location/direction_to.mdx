```py
def direction_to(location: Location) -> Direction
```

## Parameters

<DocEntry 
  entries={[

## Returns

The cardinal direction pointing towards the target location.

## Example

```py
my_loc = get_location()
target = Location(4, 7)

direction = my_loc.direction_to(target)

log(f"Move {direction} to reach the target at {target}")
```
