---
title: dig
description: Dig rubble at the agent's current location
---

```py
def dig() -> None
```

## Description

If called when no rubble is present at the agent’s current location, this function
has no effect but will still increase the agent’s action cooldown.

## Raises

<DocEntry 
  entries={[
    {
      name: "AgentError",
      description: "If digging is invalid according to game rules."
    }
  ]} 
/>

## Example

```py
loc = get_location()
top_layer = get_cell_info_at(loc).top_layer

if isinstance(top_layer, Rubble):
  dig()
```
