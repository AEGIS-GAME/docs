---
title: rotate_right
description: Rotate the direction 45 degrees clockwise
---

```py
def rotate_left() -> Direction
```

## Description

`Direction.CENTER` returns itself.

## Returns

The direction rotated right.

## Example

```py
direction = Direction.NORTH
right_direction = direction.rotate_right() # Direction.NORTH_WEST

log(f"{direction} rotated right is {left_direction}")
```
