send_message

def send_message(message: str, dest_ids: list[int]) -> None

Send a message to specified destination agents on the same team, excluding self.

Arguments

  • message (str): The content of the message to send.
  • dest_ids (list[int]): List of agent IDs to send the message to. If empty, message is broadcast to team excluding self.

get_cell_info_at

def get_cell_info_at(loc: Location) -> CellInfo

Return the cell info at a given location. If the location is adjacent (1 tile away) to the agent, or has been scanned by a drone, all layers and visible agents. If the location is not adjacent or hasn't been scanned, only the top layer is returned, and agent presence is hidden.

Arguments

  • loc: The location to query.

Returns

The `CellInfo` at the specified location, potentially with limited information depending on visibility rules.
